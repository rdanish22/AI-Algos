# AI-Algos

This repository contains the implementation of intelligent agents capable of learning, reasoning, and interacting with dynamic environments. The projects bridge the gap between classical AI (search and logic) and modern approaches (Reinforcement Learning and Deep Learning).

Key Technical Projects
The following projects demonstrate core AI competencies implemented in Python:

Heuristic Search & Planning: Implemented A Search* and custom heuristics to solve path-finding and state-space navigation problems.

Adversarial Game Playing: Developed agents for multi-agent environments (e.g., Pacman) using Minimax with Alpha-Beta Pruning and Expectimax for utility-based decision-making.

Probabilistic Reasoning (Bayesian Networks): Built a "Ghostbusters" inference engine using Bayesian Networks, Markov Models, and Inference by Sampling to locate hidden agents under uncertainty.

Reinforcement Learning (RL): Designed agents that learn optimal policies through Q-Learning, Markov Decision Processes (MDPs), and approximate Q-Learning for high-dimensional states.

Classification & Deep Learning: Architected a digit-recognition system using Perceptrons, Naive Bayes, and Deep Neural Networks (DNNs) for high-accuracy categorization.
